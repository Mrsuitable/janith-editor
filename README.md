# Janith Pandey Editor Portfolio

One-page portfolio website for Janith Pandey, a video editor working with CapCut, Adobe After Effects, Photoshop, AI editing, captions, voiceovers, and short-form content.

## Files

- `index.html` - main website
- `styles.css` - visual design and responsive layout
- `script.js` - navigation and reveal interactions
- `assets/` - portfolio photos
- `dist/index.html` - self-contained deploy file with embedded assets

Open `index.html` directly in a browser to preview the site locally.
